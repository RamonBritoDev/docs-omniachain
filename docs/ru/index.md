  <h1>Омниячейн</h1>

Среда Python для агентов ИИ — асинхронный, мультимодальный, собственный MCP.

<div class="сеточные карты" уценка>

- :rocket: **Async-first** — `asyncio` для всего, отсутствие блокировок
- :art: **Мультимодальный** — текст, PDF, изображения, аудио, видео, CSV, URL.
- :robot: **5 провайдеров** — Anthropic, OpenAI, Groq, Ollama, Google.
- :shield: **PGP Security** — Контроль доступа с помощью криптографических ключей.
- :jigsaw: **Native MCP** — Протокол контекста модели от Anthropic
- :busts_in_silhouette: **Мультиагент** — Супервизор, ReAct, Планировщик

</div>

## Быстрый старт

``` питон
из агента импорта omniachain, Anthropic, калькулятора, web_search

агент = Агент(
    поставщик = Антропный (),
    инструменты = [калькулятор, web_search],
)

result = await Agent.run("Сколько будет 1547 × 32 + √144?")
print(result.content) # "49,516"
```

**3 строки. Никакого шаблона. Готовый.**

## Почему OmniaChain?

| против Лангчейна | ОмнияЧейн |
|---|---|
| Смешанная синхронизация/асинхронность | **100% асинхронность** |
| ~20 строк на одного агента | **3 строки** |
| Нет поддержки MCP | **Встроенный MCP** |
| Нет безопасности | **Полная версия PGP** |
| Нет видеоанализа | **Рамки + Транскрипция** |
