## 支持的类型

|扩展|装载机|它有什么作用 |
|----------|------------|------------|
| `.pdf` | PDF加载器|使用 PyPDF 提取文本 |
| `.png/.jpg/.webp` |图像加载器 |用于本机视图的 Base64 |
| `.mp3/.wav/.ogg` |音频加载器 |使用 Whisper 进行转录 |
| `.mp4/.avi/.mkv` |视频加载器 | **框架+转录** |
| `.csv/.xlsx` | CSV加载器|熊猫：数据+统计|
| `.py/.js/.ts` |代码加载器 |使用信息语法的代码 |
| `http://...` |网址加载器 |用 BeautifulSoup 刮擦 |
| YouTube 网址 |视频加载器 |下载+帧+音频|

## 视频：3 层

`VideoLoader` 是独一无二的——没有任何框架可以做到这一点：

1. **📸关键帧**：提取N个分布式帧→base64→模型看到
2. **🎵音频**：提取曲目→耳语转录
3. **📊元数据**：持续时间、分辨率、编解码器、FPS
