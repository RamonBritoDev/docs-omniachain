# 代理

OmniaChain 提供 **6 种专业代理**，均具有工具调用和记忆功能。

＃＃ 概述

__代码_块_0__

|代理|专业|何时使用 |
|--------|--------------|----------|
| `代理` |一般|简单的任务 |
| `ReActAgent` |理性+行动|搜索，链式推理 |
| `多式联运代理` |任意输入| PDF、图像、音频、视频 |
| `PlannerAgent` |计划→执行→审查|复杂任务的步骤 |
| `SupervisorAgent` |协调分代理|具有委托的多代理 |
| `语音代理` | STT → 法学硕士 → TTS |语音聊天 |
| `艺术家代理` |生成图像 |使用优化的提示创建图像 |
