# MCP——模型上下文协议

OmniaChain 对 Anthropic 的 MCP（服务器、客户端和传输）具有**原生**支持。

##什么是MCP？

MCP 是一种标准化协议，用于通过 JSON-RPC 公开工具、资源和提示。允许**任何代理**从**任何服务器**访问工具。

__代码_块_0__

## 何时使用

|场景 |解决方案 |
|---------|---------|
|同一进程中的工具| `@tool` 装饰器 |
|代理之间共享的工具 | **MCP 服务器** |
|与克劳德桌面连接| **MCP 服务器（stdio）** |
|访问外部服务器工具 | **MCP 客户端** |
