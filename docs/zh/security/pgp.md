### 访问规则

|方法|效果|
|--------|--------|
| `授予（fp，工具= [...]）` |允许特定工具|
| `授予（fp，内存=[...]）` |允许内存操作 |
| `授予（fp，all_resources = True）` |允许**一切** |
| `拒绝(fp, 工具=[...])` |块工具（优先！）|

！！！警告“拒绝>允许”
    “拒绝”规则**始终**优先于“授予”。
