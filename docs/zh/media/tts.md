# 文本转语音 (TTS)

具有多个后端的语音合成 - 包括 100% 免费选项。

## 后端

|后端 |类型 |需要 |成本|品质 |
|--------|------|--------|:-----:|:---------:|
| `边缘` ⭐ |免费| `pip install edge-tts` | **免费** | ⭐⭐⭐⭐ |
| `openai` |应用程序接口 | `OPENAI_API_KEY` | $0.015/1K 字符 | ⭐⭐⭐⭐⭐ |
| `科基` |地点 | `pip 安装 TTS` | **免费** | ⭐⭐⭐ |
| `谷歌` |应用程序接口 | `pip install google-cloud-texttospeech` | 4 美元/100 万个字符 | ⭐⭐⭐⭐ |
| `自动` | — |自动检测| — | — |

！！！提示“推荐”
    **Edge TTS** 是最佳选择 — 品质卓越、免费、无 API 密钥、PT-BR 中的语音。
