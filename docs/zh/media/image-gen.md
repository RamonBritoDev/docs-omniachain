# 图像生成

使用任何 API 生成图像 - DALL-E、Google Nano Banana、Stability AI、本地稳定扩散或任何自定义后端。

## 后端

|后端 |类型 |需要 |型号|
|--------|------|--------|---------|
| `openai` |应用程序接口 | `OPENAI_API_KEY` |达尔-E 3、达尔-E 2 |
| `谷歌` / `纳米香蕉` |应用程序接口 | `GOOGLE_API_KEY` |双子座形象（纳米香蕉）|
| `稳定性` |应用程序接口 | `STABILITY_API_KEY` | SDXL、SD3 |
| `舒适` |地点 | ComfyUI 运行 |任何 .safetensors 模型 |
| `自动` | — |自动检测| — |
